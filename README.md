# AIorNot

AIの利用が適しているかを、簡単なキーワード判定で案内する単一HTMLアプリです。
プロンプト生成は行わず、判断の前提整理と注意喚起に特化しています。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「困りごと・やりたいことを自由に書いてください」に内容を入力します。
3. 「判定する」を押すと、判定結果と検出理由が表示されます。

## 判定ロジック

- AIが不向きな指標
  - 例: 「毎回同じ」「完全一致」「正確な計算」「自動処理」「ミスが許されない」
  - 「削除」「消去」など、取り消しにくい操作も注意対象

- AIが向きやすい指標
  - 例: 「考えたい」「アイデア」「整理したい」「言語化」「比較したい」

- 両方が含まれる場合
  - AIの利用が適している判定を出しつつ、強い警告を表示します。

## 出力

- AIが不向き: 決定的ツール（Python/Excel等）を推奨
- AIが向きやすい: 思考の相棒として使う旨とガイドを表示
- 条件付き注意: 一部のみAI向きで人間の検証が必須
- 情報不足: 目的と精度を追記するよう促す

## ファイル

- `index.html` 単一ファイル（HTML/CSS/JSを内包）
